    ITS131L

    Course Summary

    We used Oracle VM Virtualbox to install Linux and MySQL during this course. We learned how to create databases and tables and insert, update, and delete records in the table. In addition, we learned how to make and read an Entity Relationship Diagram (ERD). Also, we used Python and CSV to import data into tables. Finally, we installed PhpMyAdmin and created databases and tables with records there.
